Dev8085 development tools
-------------------------

This is where all the development tools will be located after
compilation.  You can install them by executing a "make install"
or you may simply copy them to your bin directory of choice.


Tool List:

	asm85	- 8085 cross assembler

	emu85 	- 8085 single board computer emulator

	dump85	- binary dump utility

	link85	- module linker

	srec85 	- Motorola S-records conversion utility


Support Files:

	emu85.cfg	- config file for emu85
